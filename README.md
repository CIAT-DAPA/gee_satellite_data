Description</h2>
<p>This repository is a collection of <strong>290+</strong> Python examples for the <a href="https://github.com/gee-community/qgis-earthengine-plugin">Google Earth Engine plugin for QGIS</a>. I developed these Python examples by converting <em>all</em> the JavaScript examples (except those not yet supported by plugin) from the Google Earth Engine <a href="https://developers.google.com/earth-engine/" rel="nofollow">API Documentation</a>. Additionally, some examples were adapted from <a href="https://github.com/gee-community/qgis-earthengine-plugin/tree/master/examples">Gena's examples</a> and the Earth Engine <a href="https://github.com/google/earthengine-api/tree/master/python/examples">API examples</a>. Kudos to <a href="https://github.com/gena">Gennadii Donchyts</a> for developing this amazing Google Earth Engine plugin for QGIS.</p>

Usage</h2>
<ul>
<li><strong>Step 1:</strong> <a href="https://earthengine.google.com/signup/" rel="nofollow">Sign up</a> for <a href="https://earthengine.google.com/" rel="nofollow">Google Earth Engine</a>.</li>
<li><strong>Step 2:</strong> Install <a href="https://qgis.org/" rel="nofollow">QGIS</a>.</li>
<li><strong>Step 3:</strong> Install the <a href="https://gee-community.github.io/qgis-earthengine-plugin/" rel="nofollow">Google Earth Engine Plugin for QGIS</a> and authenticate Google Earth Engine.</li>
<li><strong>Step 4:</strong> Git clone or <a href="https://github.com/giswqs/qgis-earthengine-examples/archive/master.zip">download</a> this repository.</li>
<li><strong>Step 5:</strong> Open the Python console in QGIS and load any downloaded Python script into the QGIS Python Editor.</li>
<li><strong>Step 6:</strong> Click the <em>Run script</em> button on the Python Editor to execute the script.</li>
<li><strong>Step 7:</strong> Zoom in/out the QGIS Canvas to inspect the results.</li>
</ul>

Add QGIS Basemaps</h2>
<ul>
<li>Open the Python console in QGIS and load the Python script (<a href="https://github.com/giswqs/qgis-earthengine-examples/tree/master/Basemaps/qgis_basemaps.py">Basemaps/qgis_basemaps.py</a>) into the QGIS Python Editor.</li>
<li>Click the <em>Run script</em> button on the Python Editor to execute the script. This will add many basesmaps as XYZ tiles to QGIS. Select and double click any basemap under XYZ Ttiles to be added to QGIS Canvas. See the screenshot below.</li>
<li>Alternatively, you can install the QGIS <a href="https://plugins.qgis.org/plugins/quick_map_services/" rel="nofollow">QuickMapServices</a> plugin. After installing the plugin, go to <em>QGIS</em> -- <em>Web</em> --<em>QuickMapServices</em> -- <em>Settings</em> -- <em>More services</em> -- <em>Get contributed pack</em> -- <em>Save</em>.</li>
</ul>
